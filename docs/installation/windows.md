# Installation on Windows
Download the .exe file from the [latest release](https://github.com/shaweel/shaweelTimer/releases/latest) and run it. Optionally make a shortcut for it to add to your Start Menu and/or Desktop.

## Creating a shortcut
Right click the .exe file and click "Create Shortcut". You can now either move this shortcut to your Desktop, or move it to C:\Users\<YOUR-USER>\AppData\Roaming\Microsoft\Windows\Start Menu which adds it to your Start Menu
